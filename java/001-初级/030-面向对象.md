# 面向对象 

## 实例化对象
```
    类名 对象名=new 类名();
    Car c = new Car();
```

## 构造器(构造方法)
### 作用：初始化类的对象,并返回对象的地址
### 示例
```
    public class Car{
        public Car(){

        }
    }
```
### 类型
+ 无参数构造器：初始化时，成员变量的数据均来源于默认值
+ 有参数构造器：初始化时，成员变量的数据为接收的参数
