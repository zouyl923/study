## 字面量
### 定义：数据在程序中展示格式
### 用法
+ 字符 必须用单引号 只能一个字符
+ 字符串 必须双引号 
+ 布尔值 true false


## 变量
### 定义：存储一个数据的，可以发生变化
### 用法
+ 格式要求： 数据类型 变量名=初始值
+ 数据类型 
```
    整数类型：
        byte:  -128 ~ 127 
        short: -32768 ~ 32767
        int: -2147483648 ~ 2147483647
        long: -9223372036854775808 ~ 9223372036854775807
    浮点数类型：
        float: 3.4x10^38
        double: 1.79x10^308
    字符类型：char
    布尔类型：boolean 
    应用型：String 
```
+ 变量名：要小写
+ 初始值
### 注意问题
+ 使用变量必须申明
+ 变量类型不可更改
+ 一个范围只能定义一次
